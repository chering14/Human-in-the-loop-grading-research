# 问题界定与相关研究

## 研究背景

作业批改是基础教育中高频、低延迟要求强、又高度依赖教师专业判断的环节。客观题和部分结构化题目适合自动化处理，但开放表达、解题过程、书写识别冲突和学生持续性错误仍需要教师理解情境。若系统仅追求“批得更多”，可能把误判风险转移给学生；若系统要求教师逐条复核，又可能将自动化收益抵消。

因此，本研究将问题界定为教师注意力配置问题，而非单纯的模型判分问题。可接受的系统目标应同时满足三点：评分可靠性不下降，教师净处理时长降低，学生获得更可执行的订正反馈。

## 外部证据梳理

教育部“人工智能+教育”相关政策将智能批改、作业管理、答疑辅导和安全可信治理同时纳入讨论范围，说明该方向具有明确政策语境，但也要求系统保留责任边界和治理机制。新华社相关调研报道显示，教师群体对人工智能工具存在较高使用率和效率期待；该类调查能够说明需求强度，却不能直接证明某一产品的学习效果。

行业侧，科大讯飞、希沃等厂商公开材料均已展示多学科批改、学情报告、课堂反馈或端云协同能力。此类材料说明供给端能力正在趋同，单纯“扫描、批改、生成报告”难以形成充分差异。更重要的差异来自教师工作流嵌入、复核证据呈现、风险分流、校本规则版本管理以及学校级数据治理。

学术研究中，形成性评价强调反馈对学习的作用不在于给出分数，而在于帮助学习者理解差距并采取下一步行动。学习分析领域的人机协同研究也提示，系统应支持教师判断，而非把教师排除在决策之外。自动作文评分和选择性分类研究进一步表明，对不确定样本选择拒答或转人工，是降低高风险误判的重要机制。

## 用户任务拆解

教师需要快速处理重复性判分，同时能看到模型不确定、证据不足或教育风险较高的样本。学生需要知道具体错误与下一步订正路径，而非只收到模板化评语。学校管理者需要可审计的指标，用于判断系统是否真正降低工作负担、是否产生群体差异、是否符合未成年人和个人信息保护要求。

## 研究假设

H1：在客观题和低风险结构化题目中，选择性自动通过可降低教师重复性处理时间。

H2：对开放题提供原作答证据、评分量表引用和教师可编辑评语，比单一分数更有助于订正完成。

H3：以置信度、题型风险和历史作答变化共同构成复核队列，比按提交时间排序更能提高教师复核的错误捕获效率。

H4：课堂表现与作业结果的联合证据可生成更有价值的教学假设，但不能替代教师对个体学生的专业判断。

## 研究问题

在不降低评分可靠性与学生权益保障水平的前提下，基于不确定性与风险分流的人机协同批改，能否降低教师净处理时长，并提高反馈后的订正质量？
